
【3Dモデル】段ボール箱 / Cardboard Box 利用規約

-------------------
VRChatのワールドやゲーム開発等に利用可能な3Dモデルです。
Unity2018 / VRCSDK2用
Substance Painterで制作したリアルなテクスチャ入り。

-------------------
【 内容物 】
UnityPackage形式
モデルfbxファイル 2種類
	通常版 12△ポリゴン
	Dirt版 48△ポリゴン

Prefab 4種類
	通常版Prefab
	通常版Prefab(坪倉研究所テクスチャ)
	Dirt版Prefab
	Dirt版Prefab(坪倉研究所テクスチャ)

テクスチャ 12枚
	ベースカラー/メタリック/ラフネス/ノーマル

マテリアル 4種類
	通常版Prefab
	通常版Prefab(坪倉研究所テクスチャ)
	Dirt版Prefab
	Dirt版Prefab(坪倉研究所テクスチャ)

-------------------
【 使用方法 】
1. UnityPackageをUnityにインポートして下さい。
2. プロジェクトフォルダ内のTsubokuLab/Models/CardboardBox/PrefabsにあるPrefabをシーンに配置して下さい。
3. シーンを再生して動作を確認して下さい。
4. VRChatのワールドとしてアップロードしたり、ゲームの背景等にご自由にお使い下さい。

※ 反射を綺麗に表現するには、ライティング及びReflection Probeの設定を行って下さい。
※ テクスチャ解像度が高めなので必要に応じてInspectorより解像度変更・画像圧縮等を行って下さい。
